# 🦅 F.A.L.C.O.N. CLI System

> Focus & Alignment Layer for Conscious Optimization & Navigation

A personal command-line tool that helps you track daily goals, focus sessions, reflections, and build discipline streaks — like your own J.A.R.V.I.S. for productivity.

## 💻 Features

- 🎯 Log top 3 daily goals
- ⏱ Track focus blocks with duration and depth
- 🧠 Reflect on wins, distractions, and improvement
- 🔥 Streak tracker for discipline or NoFap
- ⚔️ GUTS Reset Protocol to recover from urges or drift

## 🛠 Built With

- Python 3
- JSON (for local data storage)
- VS Code / GitHub CLI
